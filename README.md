# Freecad_Tutorial

This initiative is based on a truly inspiring event.

A 74 year old man asked to teach him CAD software. He is living with Parkinson’s disease, having very limited computer knowledge, and holding a diploma completed back in the 1980s, he still carries a dream to develop a machine he has been thinking about for years.

Despite the challenges, the curiosity and determination to learn remain strong.
It is a powerful reminder that the desire to learn does not age.

It also reflects a simple truth:
Sharing knowledge is one of the greatest things a person can do in life. Humanity itself is built upon sharing.

There is also a famous thought by Richard Feynman:
“If you can't explain something to a five year old, you don't understand it yourself.”

Inspired by that idea, a very simplified FreeCAD workbook is now being created so that anyone can start learning CAD from the basics.

If it can help one determined learner, it can help many more.
Anyone interested in learning, contributing, or supporting this effort is warmly welcome.
